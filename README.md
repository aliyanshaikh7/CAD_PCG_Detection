# CAD_PCG_Detection

Detecting Coronary Artery Disease from PCG heart sounds.

## Dataset

This project uses the PhysioNet/Computing in Cardiology Challenge 2016 dataset.

**Download the data:**
- Source: https://physionet.org/content/challenge-2016/1.0.0/
- Training data: `training.zip` (focusing on training-b and training-e)
- Validation data: `validation.zip`

Place the unzipped data in your Google Drive at:
`/MyDrive/CAD_PCG_Detection/CAD_PCG_Dataset/`

## Usage

1. Mount Google Drive in Colab
2. Run the notebook: `CAD_PCG_Detection.ipynb`
3. Follow the cells in order
